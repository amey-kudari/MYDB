MYDB Documentation

Mydb was designed specifically for our purposes and is highly flexible (it is extremely easy to add features to mydb) and
extremely fast (about 10x faster than postgre for database size lesser than 4gb).

Add row to table

Adding a row to a new table name creates a table

We make a post request to /query/tablename. Here, tablename is the name of the table we want to insert in

Python request

requests.post(url+'query/qwer', data = {'row' : json.dumps([1,2,'asdf'])})

Creates a table qwer if it doesnt exist and adds the row to it

Get rows of a table

We make a get request to /query/tablename. Here, tablename is the name of the table we want to query

Python request

requests.get(url+'query/qwer').content

this code returns [["1", "4", "asdf"]]

Delete a table

We make a delete request to /query/tablename. Here, tablename is the name of the table we want to delete

Note: Delete makes the table blank, hence, we use use delete to create a new empty table

Python request

requests.delete(url+'query/qwer')

This code returns [["1", "4", "asdf"]]
